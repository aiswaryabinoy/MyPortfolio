About the Project

The Smart Office Assistant is an AI-driven system that automates routine office operations. It uses facial recognition to mark employee attendance and greet them by name. For unrecognized visitors, it acts as a virtual receptionist that handles appointment booking and responds to queries through a chatbot interface. The system includes an intuitive admin dashboard, real-time monitoring, and data logging features that improve efficiency, enhance security, and reduce manual effort in modern workplaces.
